# gitflow-example

1. Создать репозиторий на gitHub и склонировать его на ПК
2. Создать ветку develop от главной ветки main 
3. создать от ветки develop/-ветки и мержить feature/-ветки в develop, когда фичи будут выполнены
4. создание ветки release/0.1.0
5. Когда ветка release/0.1.0 закончина, то она мержется в develop and main и затем удаляется
6. Если в ветке main щбноруживается ошибка, то создается hotfix-ветка.
7. когда работа завершина hotfix-ветка мержится в develop and main
